# Basketball Scoreboard (Python + Pygame)

## 실행 방법
```bash
python3 scoreboard.py
```

## 주요 단축키
- A/` : TEAM A +1 / -1
- K/- : TEAM B +1 / -1
- 1/2/3 : TEAM A +1/+2/+3
- 8/9/0 : TEAM B +1/+2/+3
- Space : 게임 시계 ▶ / ⏸
- S : 샷클락 ▶ / ⏸
- R : 전체 리셋
- [ / ] : 쿼터 -/+
- PgUp/PgDn : 게임 시간 +60/-60s
- ; / ' : 샷클락 +5/-5s
- T/Y : TEAM A/B 타임아웃 -1
- F/G : TEAM A/B 타임아웃 +1
- J : 마우스 숨김/표시
- Ctrl+T : 팀 교체
- F11 : 전체화면 ↔ 창모드
- Esc : 종료

## 실행 옵션
```bash
python3 scoreboard.py --windowed --size 1280x720
python3 scoreboard.py --teamA "벼락" --teamB "팀로이" --game 600 --shot 24 --periods 4
```
